# @notebooklm-client/notebooklm-linux-arm64

Platform-specific binary for [notebooklm](https://github.com/missdeer/notebooklm-client) (Linux ARM64).

## Overview

This package contains the pre-built binary for Linux ARM64. It is automatically installed as an optional dependency when you run `npx notebooklm` on Linux ARM64 systems.

**Do not install this package directly.** Instead, use:

```bash
npx notebooklm --help
```

## Supported Platforms

- Linux arm64/aarch64
