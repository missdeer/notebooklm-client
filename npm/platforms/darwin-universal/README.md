# @missdeer/notebooklm-darwin-universal

Platform-specific binary for [notebooklm](https://github.com/missdeer/notebooklm-client) (macOS Universal - x64 and arm64).

## Overview

This package contains the pre-built binary for macOS (Universal). It is automatically installed as an optional dependency when you run `npx notebooklm` on macOS.

**Do not install this package directly.** Instead, use:

```bash
npx notebooklm --help
```

## Supported Platforms

- macOS x64 (Intel)
- macOS arm64 (Apple Silicon)
