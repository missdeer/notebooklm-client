# @missdeer/notebooklm-linux-x64

Platform-specific binary for [notebooklm](https://github.com/missdeer/notebooklm-client) (Linux x64).

## Overview

This package contains the pre-built binary for Linux x64. It is automatically installed as an optional dependency when you run `npx notebooklm` on Linux x64 systems.

**Do not install this package directly.** Instead, use:

```bash
npx notebooklm --help
```

## Supported Platforms

- Linux x64/amd64
